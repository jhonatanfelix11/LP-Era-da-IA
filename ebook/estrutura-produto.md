### Capítulo 6 — O Fluxo de Trabalho do Profissional I.A.
**Objetivo:** Mostrar como combinar ferramentas em um fluxo real de produção
**Conteúdo:**

**Fluxo para criação de vídeo publicitário:**
1. Brief do cliente → Claude (roteiro e copy)
2. Storyboard → Midjourney (imagens de referência)
3. Geração de cenas → Higgsfield.ai ou Runway ML
4. Enhancement → Magnific.ai (qualidade final)
5. Edição → CapCut AI ou Premiere Pro + plugins AI
6. Entrega → cliente satisfeito

**Fluxo para criação de eBook/produto digital:**
1. Pesquisa → Gemini (análise de mercado)
2. Estrutura → Claude (arquitetura de conteúdo)
3. Escrita → ChatGPT + Claude (capítulos)
4. Design → DALL-E 3 + Canva AI (visual)
5. Publicação → Hotmart/Kiwify

**Fluxo para designer freelancer:**
1. Brief → Claude (entender necessidade do cliente)
2. Conceitos → Midjourney (variações visuais)
3. Enhancement → Magnific.ai (qualidade profissional)
4. Edição final → Photoshop + Adobe Firefly
5. Entrega e iteração

**Entregável:** Fluxogramas visuais dos 3 fluxos principais
