## FLUXO DE PRODUÇÃO

### 1. Gerar Imagens
Copie os prompts de `imagens/prompts-imagens.md` e gere com:
- **Imagen 4** (via Google AI Studio)
- **Midjourney** (via Discord)
- **Ideogram** (para textos nas imagens)

### 2. Animar com Kling
Para cada imagem gerada, use o prompt correspondente de `videos/prompts-kling.md`:
- Acesse [klingai.com](https://klingai.com)
- Faça upload da imagem gerada
- Cole o prompt de animação
- Exporte em 9:16 (Reels) ou 1:1 (Feed)
- Duração: 5–10 segundos

### 3. Configurar Anúncio
Copie o texto de `copies-facebook-ads.md`:
- **Headline:** texto curto (máx. 40 caracteres para Reels)
- **Primary Text:** copy completa
- **CTA:** link para landing page ou checkout
- **Orçamento sugerido:** R$ 30/dia por criativo

## RECOMENDAÇÕES

### Formatos
- **Reels/Stories (9:16):** Prioridade — maior alcance orgânico e pago
- **Feed (1:1):** Complemento — melhor para retargeting
- **Carousel:** Combine 2–3 criativos do mesmo ângulo

### Segmentação
- **Idade:** 22–38 anos
- **Interesses:** Empreendedorismo, marketing digital, tecnologia, inteligência artificial, ChatGPT, Midjourney
- **Comportamento:** Engajamento com conteúdo de negócios, freelancers
- **Localização:** Brasil (SP, RJ, MG, RS como prioritários)

### Testes A/B
- Comece com os ângulos 01 (Urgência) e 05 (Curiosidade) — testes mostram que esses convertem melhor para produtos de entrada
- Teste cada criativo por 3–5 dias antes de julgar performance
- Métricas principais: CTR > 1%, CPC < R$ 2, CPL < R$ 8

### Sequência Recomendada
1. **Semana 1:** Teste criativos 01, 05, 09 (um de cada ângulo)
2. **Semana 2:** Pause os piores, teste 03, 07 (transformação)
3. **Semana 3:** Escale os vencedores, teste 02, 10 (variados)
4. **Semana 4:** Retargeting com os melhores + criativo 08 (prova visual)
